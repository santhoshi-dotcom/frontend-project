# RePlate -- Zero Waste Food Donation System

## Project Overview

RePlate is a web-based platform designed to reduce food waste by
connecting people who have surplus food with those who are in need. The
system allows users to donate extra food through an online form so that
volunteers or organizations can collect and distribute it to hungry
people.

The project aims to create awareness about food wastage and encourage
responsible food sharing within communities.

------------------------------------------------------------------------

## Objectives

-   Reduce food waste in households, restaurants, and events
-   Provide food to people who are hungry or in need
-   Create awareness about responsible food consumption
-   Build a simple platform for food donation using web technologies

------------------------------------------------------------------------

## Key Features

-   Interactive homepage with awareness message
-   Food donation form for submitting surplus food details
-   Phone number validation
-   Food type selection (Vegetarian / Non-Vegetarian)
-   Local storage to save donation data
-   Floating food emoji animation for visual engagement
-   Simple and user-friendly interface

------------------------------------------------------------------------

## How It Works

  # 1.  The user opens the website and sees an awareness message about food
    waste.
    ![image_alt]()
  # 2.  The user clicks the **Donate Food** button.
  # 3.  A form appears asking for food details such as:
    -   Donator name
    -   Phone number
    -   Address
    -   Food name
    -   Food type
    -   Quantity
    -   Food condition
4.  The user submits the form.
5.  The donation data is stored in the browser using localStorage.

## Social Impact

Food waste is a major global issue while millions of people suffer from
hunger every day. RePlate encourages people to donate excess food
instead of throwing it away, helping to reduce hunger and food wastage.
